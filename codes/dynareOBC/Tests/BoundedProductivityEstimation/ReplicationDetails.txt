This folder contains the estimation examples.

Note that at present the estimation code is not in a good state, so it is not recommended that you use it.

The data files can be regenerated by running BoundedProductivityDataGeneration and UnboundedProductivityDataGeneration.

To run the estimation(s), run:

dynareOBC UnboundedProductivityEstimation estimation [OrderOverride=1]
dynareOBC UnboundedProductivityExactEstimation estimation DataFile=UnboundedProductivityEstimation.xlsx [FilterCubatureDegree=9] [DynamicNu]
dynareOBC UnboundedProductivityEstimation estimation DataFile=BoundedProductivityEstimation.xlsx FixedParameters=beta [OrderOverride=1]
dynareOBC UnboundedProductivityExactEstimation estimation DataFile=BoundedProductivityEstimation.xlsx FixedParameters=beta [FilterCubatureDegree=51]
dynareOBC BoundedProductivityEstimation estimation [OrderOverride=1] [FixedParameters=beta]
dynareOBC BoundedProductivityEstimation estimation fastcubature OrderOverride=1 [FixedParameters=beta]
dynareOBC BoundedProductivityExactEstimation estimation DataFile=BoundedProductivityEstimation.xlsx [FilterCubatureDegree=9] [DynamicNu] [FixedParameters=beta]
